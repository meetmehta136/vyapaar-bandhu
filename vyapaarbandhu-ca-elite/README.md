# VyapaarBandhu — CA Dashboard
> React frontend for VyapaarBandhu GST Compliance Platform

[![Live](https://img.shields.io/badge/Dashboard-Live-blue)](https://vyapaarbandhu-ca-elite.vercel.app)

## Features
- 8 pages: Dashboard, Clients, Invoices, AI Insights, Analytics, Alerts, Admin, Settings
- AMOLED dark theme
- Real-time data from FastAPI backend
- Add clients, approve/reject invoices, download filing PDFs

## Stack
React + TypeScript + Tailwind + Recharts + Vercel
